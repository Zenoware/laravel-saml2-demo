# Sailstream

## Overview

Clone this starter project if you're in a hurry to spin up a Laravel project with Sail + Jetstream for the UI scaffold.

## Requirements

Docker

## Installation

### Clone

    git clone git@github.com:Zenoware/sailstream.git

### Run

    cd sailstream
    ./vendor/bin/sail up

### Copy env

    cp .env.example .env

### Migrate

    ./vendor/bin/sail artisan migrate
